# Explain-While-Building

## Objetivo
Mantener al usuario (No Programador) informado y en control.

## Reglas
1. **Lenguaje Simple**: Usar español claro, analogías de mapa mental. No jerga académica.
2. **Contexto**: Explicar el "Por qué" antes del "Cómo".
3. **Transparencia**: Narrar las decisiones mientras se toman.
4. **Educativo**: Ayudar al usuario a entender su propio sistema sin abrumarlo.
