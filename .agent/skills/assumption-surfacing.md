# Assumption Surfacing (Explicitación de Supuestos)

## Objetivo
Evitar errores costosos basados en malentendidos.

## Reglas
1. **Identificar**: Detectar qué cosas estoy asumiendo (e.g., "El usuario tiene Node instalado", "La base de datos está activa").
2. **Validar**: Preguntar o verificar antes de construir sobre un supuesto.
3. **Documentar**: Listar los supuestos críticos en los planes de implementación.
